# Pokémon Explorer UI

A React-based interface to explore and filter structured data through a clean, interactive UI.

## Live Demo
https://chiragsomashekar.github.io/pokemon-explorer-ui/

## What it does
- Filter data dynamically by category (type)
- Mark items as favorites to simulate user preference
- Present structured data in a clear, card-based layout

## Why I built this
To bridge data and user experience by building an interface that simulates how users explore, filter, and interact with structured datasets.

## Tech Stack
- React
- Vite
- JavaScript
- CSS
